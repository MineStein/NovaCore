Welcome to NovaCore.

NovaCore is a project developed by MineStein. It aims to make mini-game development quicker, more efficient, and bug-free. If you wish to contribute to this project, contact MineStein on Skype at: minestein3646.

Getting started

To get started using this awesome development tool, start by adding the following API's to your libraries:

  • BukkitAPI
  • BarAPI

Once you have these two installed, you are ready to begin! Create your main class (don't name it "Core"!). This classes' parent will not be JavaPlugin, instead it will be Core from the com.minestein.novacore package.

This is what it should look (somewhat) like:

 public class Main extends Core {
     @Override
     public void onLoad() {

     }
     @Override
     public void onEnable() {

     }
 }

Then inside of your plugin.yml file, add the following:

depend: [NovaCore]

Some Basics

    setPrefix(String);
    setSoftPrefix(String);
    setDefaultScoreboardEnabled(Boolean);

NOTE: You must set these three in the onLoad for them to work properly!

setPrefix and setSoftPrefix are used to change the overall feel of messages from the server. The ordinary prefix is the one with brackets around it. For example: "[ExampleGame]". The soft prefix is used in the scoreboard (if the default one is enabled) and bar messages. It is the one in random colors without the brackets. For example: "ExampleGame".

setDefaultScoreboardEnabled will tell whether or not the server uses the default side scoreboard in NovaCore. This includes a nice rainbow interface using the soft prefix with a random color code from the valid colors as the objective.
